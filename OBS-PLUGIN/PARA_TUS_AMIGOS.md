# 🎵 ¡Hola! ¿Quieres instalar el Spotify Overlay?

## Para tus amigos - Instrucción Súper Fácil

### 📥 Si tienes el instalador (.exe):

1. **Descarga** el archivo `SpotifyOverlay-Setup.exe`
2. **Ejecuta** el instalador
3. **Sigue** las instrucciones
4. **¡Listo!** Abre OBS y disfruta

---

### 📦 Si tienes la carpeta completa:

1. **Abre la carpeta** del plugin
2. **Doble click** en `install.bat`
3. **Espera** a que termine
4. **¡Listo!** Abre OBS

---

### 🔧 Si lo anterior falla:

1. Abre PowerShell en la carpeta
2. Escribe: `.\install.ps1`
3. Presiona Enter
4. Sigue las instrucciones

---

## 🎮 ¿Cómo se usa en OBS?

1. **Abre OBS Studio**
2. En **Fuentes**, click en **+**
3. Elige **Navegador** (Browser)
4. Pon de nombre: `Spotify Overlay`
5. En URL escribe: `http://localhost:9274/`
6. Ancho: `450`, Alto: `150`
7. Click en **Aceptar**
8. ¡Reproduce música en Spotify!

---

## ❓ ¿Problemas?

**No aparece nada:**
- Asegúrate de que Spotify esté reproduciendo
- Reinicia OBS
- Revisa que el plugin esté instalado

**Error al instalar:**
- Ejecuta como administrador
- Revisa que OBS esté instalado

**¿Necesitas ayuda?**
- Revisa el archivo `INSTALAR.md`
- O contacta a quien te pasó el plugin
