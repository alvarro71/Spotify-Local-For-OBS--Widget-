# Plugin Icons Directory

Place the plugin icon here.

## Required File

- `plugin.png` - 128x128 or 256x256 PNG icon for the OBS source selector

## Design Guidelines

- Should be recognizable at small sizes (24x24 in OBS source list)
- Use the Spotify green (#1DB954) as the primary color
- Simple silhouette or icon (music note, headphones, or the overlay card)
- Transparent background preferred
